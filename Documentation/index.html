J) How To Install and Use SASS With CMD (Widnows User only) - top

  • Step 1: Download Ruby From Here and Install It Download Ruby (Ruby 2.3.3 Is Better)
  • Step 2: After finishing install, open your CMD with Administrator privilege and Type "gem install sass" (without quote)
  • Step 3: If your sass installed properly check your sass version with this command "sass -v" (without quote)
  • Step 4: Now You are enabled to write sass and compile sass from your CMD

How To Use SASS With CMD (Widnows User only)

  • Step: 1 Open Command Prompt and type - "sass --watch/path of sass folder/style.scss:destination path/style.css"
